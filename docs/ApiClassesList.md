# External API

- Histogramator
- SourceFlowGenerator

# Internal API
- ConfigurationParser
- ChunkAggregator
- ChunkStorage
- ChunkQueue
- ChunkParameterBuilder

## Histogramator

- Получает список чанков и аккумулирует их в общую гистограмму;
- Устанавливает границы гистограммы из конфигурации;
- При старте использования при помощи `ChunkParameterBuilder` настраивает тип данных из источника и параметры чанков, такие как:
    - число чанков;
    - размер каждого чанка;

## SourceFlowGenerator

- Генерирует поток исходных данных для построения гистограммы;
- Вынесен во внешнее API, для возможности использования различных типов данных при построении гистограммы.\
  Реализация потока должна быть осуществлена пользователем через имплементацию интерфейса.
